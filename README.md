A web-based platform that connects customers with professional cleaning service providers. **Scourgify** allows users to request house cleaning services, while cleaners can manage their profiles, view assigned cleaning requests, and track upcoming jobs through a centralized dashboard.

---

## 📌 Project Overview

Scourgify is designed to simplify the process of booking and managing house cleaning services. Customers can submit cleaning requests by providing details such as the location, address, and preferred service, while registered cleaners can browse or receive assigned requests and manage their scheduled jobs.

The project demonstrates the development of a full-stack web application with user authentication, role-based functionality, database management, and responsive web design.

---

## 🎯 Objectives

* Provide an easy way for customers to book house cleaning services
* Allow cleaners to create and manage professional profiles
* Enable cleaners to view and manage assigned cleaning requests
* Maintain customer and booking information securely
* Deliver an intuitive and user-friendly experience for both customers and service providers

## 🚀 Run the Project Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/NancyMerciline/Scourgify.git
```

### 2️⃣ Create a virtual environment

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
flask run
```

Open your browser and navigate to:

```text
http://127.0.0.1:5000/
```
